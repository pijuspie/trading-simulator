# trading-simulator

Pages:
* Without authorization
  * List of stocks
  * Details of a stock
* With authorization
  * Sign up
  * Log in
  * Profile
  * Groups
  * Overview of a group
  * Group settings
  * Buy/sell stocks in a group


### Set up

```
python3 -m venv env
env\Scripts\activate
python3 -m pip install yfinance
python3 -m pip install -r requirements.txt
python3 main.py
```
